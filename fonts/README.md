# Optional CJK fonts for PDF

Place an open-licensed font here to embed Chinese in PDFs on Streamlit Cloud
without a first-run download:

- `NotoSansSC-Regular.otf` (preferred, OFL)
- `NotoSansSC-Regular.ttf`

If missing, the app tries system fonts, then downloads Noto Sans SC to a temp cache.
Do not commit proprietary Windows fonts (e.g. SimHei / Microsoft YaHei).
