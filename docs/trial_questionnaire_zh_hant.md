# Sigma Fate 试用问卷（15 题）

**试用链接：** https://sigma-bazi.streamlit.app/  
**推荐方式：** 登录 App → 顶部 **「📝 试用反馈」** 页直接填写（自动保存，无需微信）  
**说明：** 除最后一题外，每题打 **1–10 分**（1=很差/完全不符，10=很好/非常吻合）

---

## App 内填写（推荐）

1. 打开 https://sigma-bazi.streamlit.app/  
2. 注册 / 登录  
3. 完成排盘并预览报告（建议）  
4. 点击顶部 **「试用反馈」** → 滑动打分 → 填写开放题 → 提交  

管理员在后台 **用户管理** 页可查看所有回复与均分。

> 数据库：请执行 `sql/009_sf_survey_responses.sql`
